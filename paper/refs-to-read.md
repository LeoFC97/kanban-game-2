# Referências para leitura — roteiro de acesso

Arquivo complementar a `bibliografia.bib`. Lista cada entrada recomendada com:

- **DOI / URL estável** (quando existe)
- **Rota de acesso gratuito legítimo** (Open Access, página do autor, preprint) ou
- **CAPES/CAFe** (acesso institucional UFF) quando só há via paga

Legenda:

- `OA` — Open Access (PDF público legal)
- `AUTOR` — cópia legal no site pessoal/institucional do autor
- `CAPES` — acesso pelo Portal de Periódicos CAPES via CAFe/UFF
- `UFF` — livro físico/e-book via Biblioteca UFF (Pergamum)
- `GRATIS` — gratuito no site oficial do editor/organização

## Como resolver pela UFF (passo a passo rápido)

1. Entre em <https://www.periodicos.capes.gov.br>.
2. Clique **"Acesso CAFe"** → escolha **Universidade Federal Fluminense**.
3. Faça login com seu IdUFF.
4. Pesquise pelo DOI ou título. Os links acima funcionam diretamente no portal autenticado.

Livros: entre em <https://bibliotecas.uff.br> → Pergamum → busca por título.

## Observações sobre esta lista

- URLs de download direto foram verificadas para o Scrum Guide (EN/PT-BR) e para os dois papers PLOS (Wilson 2017, Sandve 2013) em 2026-04-15.
- Demais URLs são baseadas em DOIs estáveis ou convenções conhecidas dos sites. Se algum link de `AUTOR` estiver fora do ar, o DOI e o `CAPES` permanecem como rota alternativa.
- Nenhum link aponta para repositórios piratas (Sci-Hub, LibGen, etc.). Acessos via `CAPES` são legais e cobertos pela sua matrícula UFF.
